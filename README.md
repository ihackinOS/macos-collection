# Tổng Hợp Bộ Cài macOS ( Có OpenCore 1.0.2 )

- Chỉnh cấu hình BIOS
  - SATA → **AHCI**
  - Disable → **Secure Boot**
  - Disable → **CSM**
  - Enable → **Above 4G Decoding**
